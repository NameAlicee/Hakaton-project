Для тестирования программы необходимо следовать следующим шагам:
1)запустить Microsoft MS SQL Studio 2019
2)Создать пустую базу Hakaton_project
3)Открыть скрипт script.sql, находящийся на гитхабе
4)выполнить Sql запрос, тем самым создав необходимую базу данных
5)Открыть HakatonProject.sln при помощи Visual Studio 2019
6)В обозревателе решений в папке models удалить текущую модель,затем в файле webConfig, находящемся в самом низу списка файлов обозревателя решений, удалить строку Connection String, не затрагивая тег </configuration>
7)добавить новую модель ADO-NET, подключив к нашей базе,назвав её HakatonProject
